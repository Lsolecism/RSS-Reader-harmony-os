# RSS Reader

An RSS feed reader application with a Spring Boot backend and Harmony OS frontend.

## Backend

Java Spring Boot 3.x application providing RSS feed parsing, user management, and article delivery.

- **Framework**: Spring Boot 3.4.6
- **Java**: 17
- **Database**: MySQL with MyBatis Plus
- **Cache**: Redis
- **Message Queue**: RabbitMQ
- **RSS Parsing**: Rome
- **Auth**: Spring Security + JWT

## Frontend

Harmony OS native application for reading RSS feeds on Harmony OS devices.
